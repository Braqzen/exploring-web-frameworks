## Overview

Ranking frameworks based on DevEx and paradigms. \
This is not an objective tier list, it has nothing to do with merit, utility, functionality, or performance.

It's currently based on vibes of usage but later will likely be replaced based on available features mixed with vibes.

Reasoning might be re-added after enough web-frameworks have been implemented and researched further as some things may just be a skill issue that hasn't been researched.
